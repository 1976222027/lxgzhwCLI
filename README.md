# lxgzhw基于Bootstrap3的脚手架使用说明

## 安装
- 1 项目十分轻量级,可以直接复制main.js内容到本地任何位置
- 2 只要有main.js 可以随处运行

## 使用依赖
- 1 依赖于node环境,最好是最新版(作者是用最新版测试的)
- 2 再无其他任何依赖
- 3 Bootstrap使用了官方CDN加速,无需额外安装(需要联网使用)

## 使用方法
- 1 命令1 `node main.js`
- 2 命令2 `node main.js 你要创建的项目名`
